Install dependencies: python, npm, node.js, etc.
* navigate into /dev
* pip install flask
* navigate into /client
* npm install

To run locally:
* navigate into /dev/client
* npm run build
* navigate into /dev
* python app.py
* Test on http://127.0.0.1:5000 in a browser
* Deployed at https://backpack-game.vercel.app/
